NFPA Inspection Log

About

Wrap Designer (AngularJS)

About

Fishing Rod Wrap Designer is a simulator to show a user their thread color combinations before production starts on the fishing rod. The app gives the user a realistic idea on the design, width and color choices made before production is started on making the fishing rod. The user can pick from real thread MFG colors and change widths to create specific thread patterns. Users can also pick handles and view a scaled down version of the entire fishing rod, that includes: wraps, blank color, and a selected handle. This is a client side app that is using AngularJS.

  • HTML5
  • CSS3 -> Flex Box
  • Javascript
  • AngularJS

  • Photoshop
