Wrapper Code

About

Wrapper Code is a platform ment for the rod building craft. The web application has a series of necessary Mathematical equations needed to completed specific decorative cross wraps. Guide Spacing Database

About

This app started out as a referance, for myself, that provides guide sizes and spacing statistics specific to ice fishing rod blanks, blank lengths and handle lengths. I started passing the URL around to fellow rod builders and the app took off from there. The app has been through several renditions and is currently more of a "log book" than a guide spacing referance. Users can now create an account, get authenticated and save their own stats in the PostgreSQL database with specific notes and file uploads. Guide Spacing DB currently has over 8,000 registered users and is used several hundred times daily.

Anna's Bananas Pre-School and Daycare

About

This app was developed to help streamline Anna's new employee's correctly fill out the New Employee document packet. The original packet consists of 62 pages where the staff would give this to the new employee to complete but is never properly completed. This app is user friendly, and guides the employee through the documentation and how to file correctly. Once the new employee has completed filling out all required fields, the app then creates a PDF file which contains the data the user provided/submitted, and is then emailed to an Anna's Bananas employee to process. This app is used daily and has been used several hundred times within the last year.

Wrap Designer (AngularJS)

About

Fishing Rod Wrap Designer is a simulator to show a user their thread color combinations before production starts on the fishing rod. The app gives the user a realistic idea on the design, width and color choices made before production is started on making the fishing rod. The user can pick from real thread MFG colors and change widths to create specific thread patterns. Users can also pick handles and view a scaled down version of the entire fishing rod, that includes: wraps, blank color, and a selected handle. This is a client side app that is using AngularJS.
